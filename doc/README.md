# meeting-whisperer (知ったかくん)

会議中に理解できていない前提知識を、その場で静かに補完する AI。

一般的な会議AIが「会議全体の要約」を目指すのに対し、このプロジェクトは「個人の理解不足補完」に特化する。

## One-liner
- 会議を止めずに、置いていかれた個人を救う「認知ギャップ補完AI」。

## Problem
- 会議中に知らない単語を聞き返しづらい
- 他部署・新人・途中参加者が会議についていけない
- 社内略語・専門用語・プロジェクト固有語が多い
- “知ったか”で乗り切るしかない

## Solution
- 会議ログから専門用語候補を抽出
- ユーザーが気になった語だけクリック
- クリック語を、会議文脈付きで短く説明
- クリック履歴を蓄積し、個人向けの理解補助を強化

## Why Now / Why Microsoft
- Azure上でサーバーレスに短期間実装可能
- Azure OpenAI / Agent系サービスで説明生成・補助文生成を実現
- Microsoft ecosystem (Teams/Entra/Graph) へ将来連携しやすい

## Hackathon Fit
- 新規性: 会議要約ではなく「個人認知ギャップ」へフォーカス
- 効率化: 会議の流れを止めずに理解を補完
- Agentらしさ: クリック履歴を学習信号として個人最適化
- 実業務適用: オンボーディング、横断PJ、略語文化の強い現場で有効

## MVP Scope
- テキスト会議ログ入力 (textarea / txt / md)
- 用語抽出
- 用語詳細説明
- クリック履歴保存
- 「今日の知ったかまとめ」生成
- knowledge profile の初期生成・更新
