# MVP Plan

## Priority
1. 会議テキスト入力
2. 用語抽出
3. 用語説明
4. クリック履歴保存
5. 今日の知ったかまとめ
6. knowledge profile 更新

## Out of Scope (Hackathon MVP)
- リアルタイム音声認識
- Teamsリアルタイム会議連携
- 複雑な認可・権限管理
- AKS / Kubernetes
- 過剰なマルチエージェント化

## Success Criteria
- 3分以内にデモ可能
- 初見ユーザーが迷わないUI
- クリックした語に対して文脈付き説明が返る
- 会議後まとめが個人観点になっている

## Demo Script (Short)
1. 会議ログを貼る
2. 候補語が出る
3. 2語クリックして説明を見る
4. まとめ生成を押す
5. 個人向け要約を提示
